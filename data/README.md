# Data Directory

Place challenge data files here when reproducing the pipeline locally.

Expected files:

- `pxr-challenge_TRAIN.csv`
- `pxr-challenge_TEST_BLINDED.csv`
- `phase1_unblinded.csv`
- `pxr-challenge_counter-assay_TRAIN.csv`
- `pxr-challenge_single_concentration_TRAIN.csv`
- `multitask_train.csv`
- `pxr_train_mmps.json`

Large challenge data files are intentionally not committed by default.
